# advanced-calculator-c
📌 DESCRIPTION:
This is a command-line based calculator written in C.
It preforms basic mathematical operations along with advanced features like multiple number calculations, loop-based execution and max-min finder.

🚀 FEATURES:
* Addition, Subtraction, Multiplication, Division
* Marks Percentage Calculation
* Multiple Number Addition, Subtraction & Multiplication
* Max-Min Finder using logic/array
* Loop-based continuous execution
* Input validation using if condition

🛠️ TECHNOLOGIES USED:
* C Programming Languagr

▶️ HOW TO RUN:
1. Compile the code: gcc calculator.c-o calculator
2. Run the program:./calculator
